# ClimaApp
Aplicación en consola en Node que permite buscar lugares en todo el mundo con su temperatura.
También conserva los ultimos 5 lugares buscados.

```
========================
 Seleccione una opción 
========================

? ¿Qué desea hacer? (Use arrow keys)
❯ 1. Buscar ciudad 
  2. Historial 
  0. Salir 
  ```
